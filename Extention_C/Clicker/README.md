# Auto Clicker
크롬 확장프로그램 입니다.

단축키를 이용하여 무한 클릭을 할 수 있어요. 저도 만들어놓고 어디다 써야할지 잘 모르겠네요.  
Dom 엘리먼트에 클릭이벤트를 발생 시킵니다. 버블링되어 상위 태그 이벤트가 발동 됩니다.
